Vào Edit HTML/CSS để code:

Cấu trúc thư mục Shopify Theme: bao gồm 6 thư mục:

Thư mục 1: Layout:

Thư mục 2: Templates:

  • - "404.liquid": error 404 Page
  • - "article.liquid": Single Post
  • - "blog.fullwidth.liquid": Trang Blog full width
  • - "blog.leftsidebar.liquid": Trang Blog có left sidebar
  • - "blog.rightsidebar.liquid": Trang Blog có right sidebar
  • - "cart.liquid": Shopping Cart Page
  • - "collection.16.liquid": Trang Collection hiển thị 16 products/1page
  • - "collection.32.liquid": Trang Collection hiển thị 32 products/1page
  • - "collection.all.liquid": Trang Collection hiển thị tất cả products/1page
  • - "collection.fullwidth.liquid": Trang Grid Collection full width
  • - "collection.fullwidth.list.liquid": Trang List Collection full width
  • - "collection.leftsidebar.liquid": Trang Grid Collection có left sidebar
  • - "collection.leftsidebar.list.liquid": Trang Grid Collection có right sidebar
  • - "collection.liquid": Trang Collection default
  • - "customers/account.liquid": Account Page
  • - "customers/activate_account.liquid": Activate account Page
  • - "customers/addresses.liquid": Addresses Page
  • - "customers/login.liquid": login Page
  • - "customers/order.liquid": order Page
  • - "customers/register.liquid": register Page
  • - "customers/reset_password.liquid": Reset password Page
  • - "index.liquid": Home Page
  • - "page.contact.liquid": Contact Page
  • - "page.liquid": Default Page
  • - "page.wishlist.liquid": Wishlist Page
  • - "password.liquid": Coming soon
  • - "product.liquid": Trang chi tiết sản phẩm
  • - "search.liquid": Search Page

3. Đóng gói

3.1 Export your theme

Go to this page http://docs.shopify.com/manual/your-website/navigation/create-drop-down-menu to know how to create dropdown menu in Shopify.

Then click Send Emai. a email will send to your email. Into your email to download theme.

3.2 Export Products

Chọn Export "All products" và kích "Export Products" button.

3.3 Export Pages

Vào page. copy toàn bộ nội dung của page lưu vào file .xml. Ex: about-us.xml

4. Hoàn thiện

Tạo folder đóng gói: tên_theme_v1.0_UNZIP_FIRST/

Ex: folder: sp_goshop_v1.0_UNZIP_FIRST/

Bao gồm các folder con: Documentation/, Licensing/, Sample Data/, Theme/

Folder Documentation:

Để trống

Folder Licensing:

Gồm 2 files: GPL.txt, README_License.txt

Folder Sample Data: Gồm file đóng gói products và các page có nội dung.

Folder Theme: Bao gồm các file đóng gói theme .zip của các home. Ex: sp-goshop-home1.zip, sp-goshop-home2.zip, sp-goshop-home3.zip ...

Cuối cùng nén folder "sp_goshop_v1.0_UNZIP_FIRST/" thành file .zip : sp_goshop_v1.0_UNZIP_FIRST.zip (Giống file .zip trong folder "FINISH/"). Gửi file này cho quản lý.
